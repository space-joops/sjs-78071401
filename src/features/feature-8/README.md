# 기능 8 — 줍스(Joops) 코어 루프 MVP

궤도 방치 + 상공 진입 액션 + 다마고치식 치료를 검증하는 싱글 플레이 프로토타입.

## 코어 루프

1. **설정**: 첫 진입 시 궤도 위 지점을 탭해 '내 상공(Home Point)'을 지정.
2. **방치(관제 화면)**: 줍스가 2분에 한 바퀴 궤도를 돌며 5초마다 확률적으로 소량 EXP 획득. 하단에 내 상공 도착 ETA 표시.
3. **액션**: 내 상공(±20°) 진입 시 버튼 활성화 → 45초간 날아오는 우주 쓰레기를 탭해 부수고 개당 6 EXP 획득. 종료 후 줍스는 상공 창을 빠져나간 지점부터 다시 비행.
4. **성장**: 레벨업 시 줍스 몸집이 커지고 색상(hue)이 변함.
5. **충돌·치료**: 비행 중 평균 3분에 1회 잔해 충돌 → HP 0, 궤도 정지. 정지한 줍스(또는 하단 버튼)를 탭해 치료 모드 진입, 5회 탭으로 완치 후 비행 재개.
6. **영속성**: `localStorage`(`sjs-feature-8-joops-save-v1`)에 저장. 재접속 시 부재 시간을 시뮬레이션(궤도 진행, 방치 EXP, 지수분포 충돌 샘플링)하고 30초 이상 부재면 '부재중 관제 보고' 모달 표시.

## 구조

- `lib/game.ts` — 상수·타입·순수 로직(실시간 틱과 오프라인 시뮬레이션이 동일 수식 공유)
- `lib/storage.ts` — localStorage 저장/복원
- `hooks/useJoopsGame.ts` — 상태 머신(setup/orbit/action/heal)과 100ms 게임 틱
- `components/OrbitView.tsx` — SVG 관제 화면(지구·궤도·줍스·내 상공)
- `components/ActionGame.tsx` — 쓰레기 줍기 미니게임 오버레이
- `components/HealOverlay.tsx` — 치료 오버레이
- `JoopsGame.tsx` — 조립 + HUD

밸런스 수치(궤도 주기, 충돌 확률, EXP 등)는 전부 `lib/game.ts` 상단 상수로 모여 있다.
